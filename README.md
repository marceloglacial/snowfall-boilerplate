# Snowfall Boilerplate

* Created by Marcelo Glacial
* Starts on nov/2018

## Description

A starter kit to new web projects based on WordPress.


## Features
End-to-end WordPress project based on three stages:

1. Design
2. Front-end
3. Back-end

## Project Instalation

```terminal
    npm install 
    npm audit fix --force
    gulp install
```
